## ProjetosAndroidTurma02Ralf

# Pasta Projeto 1 

- Introdução ao Desenvolvimento com Kotlin -> Feito um projeto voltado para um sistema de Pessoas. Cadastrar, remover, listar, alterar.

# Pasta gerenciardespesas (Projeto 2) 

- Aplicativo para gerenciar as despesas pessoais. Usando MVVM, StateFlow, Bottom bar. Funcionalidades: O usuario poderá cadastrar Ganhos, Gastos, Sonhos/Desejos. E possue uma Página inicial, Exibindo o valor em conta.

# Pasta aplicacaocepmvvm (Projeto 3) 

- Aplicativo para adicionar contatos. Usando MVVM, a funcionalidade principal: adicionar informações do contato e adicionar na lista, inclue, nome, email, telefone, nascimento, CEP ( e quando for inserido o CEP, vai buscar informações na API https://viacep.com.br/, e auto completa as informações de Cidade, Estado, Bairro e Logradouro) e Número.
